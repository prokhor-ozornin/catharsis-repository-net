static IRepository<ENTITY> Catharsis.Repository.IRepositoryExtensions.Transaction< ENTITY > ( this IRepository< ENTITY >  repository,
Action  action 
)
static

Wraps a set of operations over repository inside an atomic transaction, making it a single unit-of-work block.

If no exceptions occurs during execution of action , transaction is succesfully committed, and all changes made inside it's body are persisted to the underlying data storage; otherwise, transaction is rolled back and all changes are discarded.
